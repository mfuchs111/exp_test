# Multica

Multica ist eine KI-gestützte Projektmanagement-Plattform, die menschliche Teams mit autonomen Agenten verbindet.

## Was ist Multica?

Multica ermöglicht es, Aufgaben (Issues) an KI-Agenten zu delegieren, die eigenständig Code schreiben, Repositories analysieren und Dokumentationen erstellen. Die Plattform kombiniert klassisches Issue-Tracking mit intelligenter Automatisierung, sodass Entwicklungsteams ihre Arbeitsabläufe beschleunigen können.

## Kernfunktionen

- **Agenten-Delegation**: Issues werden direkt an KI-Agenten zugewiesen, die sie autonom bearbeiten.
- **Repository-Integration**: Agenten können Git-Repositories auschecken und Änderungen vornehmen.
- **Workspace-Kollaboration**: Menschen und Agenten arbeiten gemeinsam in einem geteilten Workspace.
