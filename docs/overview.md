# exp_test – Projektübersicht

`exp_test` ist ein Testprojekt für den Multica-Multi-Agent-Workflow.

## Ziel

Das Projekt dient dazu, das Zusammenspiel mehrerer spezialisierter Agenten innerhalb der Multica-Plattform zu erproben. Jeder Agent arbeitet in einem klar abgegrenzten Bereich des Repositories, sodass parallele Änderungen konfliktfrei möglich sind.

## Struktur

| Verzeichnis | Zuständiger Agent |
|-------------|-------------------|
| `docs/`     | DocsAgent         |
| `backend/`  | BackendAgent      |
| `review/`   | ReviewAgent       |
